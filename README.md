# Bike Purchase Analysis with Excel

## Project Overview
This project analyzes customer bike purchasing behavior using Microsoft Excel. The entire workflow data cleaning, analysis, and visualization was performed exclusively in Excel to demonstrate strong spreadsheet-based analytical skills.


## Dataset used
<a href = "https://github.com/charles-tabi-gyansah/Data-Analysis-Dashboard/blob/main/Bike_Purchase%20Analysis.xlsx" >Dataset</a>


## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards
- Excel Functions (IF, VLOOKUP/XLOOKUP, COUNTIFS, AVERAGEIFS)


## Dataset Description
The dataset contains customer demographic and lifestyle information such as age, gender, income, education level, marital status, region, and whether the customer purchased a bike.


## Data Cleaning Process
- Removed duplicate records
- Standardized column names for clarity
- Corrected inconsistent categorical values
- Created new calculated columns (e.g., Age Group)
- Validated numeric fields for accuracy


## Analysis Performed
- Customer purchase behavior by gender and age group
- Income trends between buyers and non-buyers
- Impact of commute distance on bike purchases
- Regional and marital status comparisons


## Visualizations Created
- Bar charts showing bike purchases by gender and region
- Line and column charts for income and age trends
- Interactive dashboard using Pivot Charts and slicers
